🚀 Este backend foi desenvolvido para dar suporte completo ao My Drugs Frontend, sendo responsável por toda a lógica de negócio, autenticação e comunicação com o banco de dados. O projeto foi criado logo após a conclusão dos meus estudos na Rocketseat, com o objetivo de colocar em prática os conhecimentos adquiridos e entender melhor como funciona a estrutura de uma aplicação real no lado do servidor.

Durante o desenvolvimento, trabalhei na organização do código seguindo boas práticas de arquitetura, separando as responsabilidades em módulos como Controllers, Services, Repositories, Routes, Schemas e Utils. Essa estrutura me ajudou a compreender melhor o fluxo de dados dentro da aplicação, desde a requisição até a resposta final para o cliente.

Além disso, implementei a comunicação com o banco de dados utilizando ORM, o que fortaleceu minha base em persistência de dados, validações e segurança. O backend também foi pensado para se integrar diretamente com o frontend, permitindo o gerenciamento de usuários e dados da aplicação de forma segura e escalável.

Esse projeto foi muito importante para consolidar meus conhecimentos em desenvolvimento fullstack, principalmente na criação de APIs, autenticação, organização de código e comunicação entre diferentes camadas da aplicação.
